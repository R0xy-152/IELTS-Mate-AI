# IELTS-Mate AI: 多模态雅思学术词汇引擎

本项目是一款基于 **Gemini 3 Flash** 与 **Veo/Nano Banana 2** 引擎构建的智能雅思备考工具。它不仅提供基础释义，更能针对学术背景进行动态语义分析。

## 🌟 核心亮点
- **动态语义识别**：利用 AI 自动识别单词的专业场景（如将 "Thermal" 识别为热力学场景）。
- **原生多模态生成**：集成 Google Veo 引擎，为每个学术词汇实时生成 100% 相关的教学视频与 4K 图像。
- **工业级后端**：基于 FastAPI 与 SQLAlchemy 构建，具备完整的数据库持久化与安全过滤逻辑。

## 🛠️ 技术栈
- **AI 核心**：Gemini 3 Flash, Veo (Video Generation), Nano Banana 2 (Image Generation)
- **后端框架**：Python FastAPI
- **数据库**：SQLLite (SQLAlchemy ORM)
- **版本控制**：Git / GitHub

## 📈 为什么做这个项目？
旨在解决雅思备考中“学术词汇脱离场景”的痛点，展示如何利用生成式 AI 技术辅助英语学习。
